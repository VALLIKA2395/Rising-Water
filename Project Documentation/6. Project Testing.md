# Project Testing

## Introduction
Testing is performed to verify that the Rising Waters application works correctly and produces accurate results.

## Testing Types

## Functional Testing
Checks whether all features such as data input, prediction, and output display work properly.

## Model Testing
The machine learning model is tested using sample data to evaluate prediction accuracy.

## User Interface Testing
The application interface is tested for proper navigation and usability.

## Test Cases

| Test Case | Input | Expected Result |
|---|---|---|
| Flood Prediction | Environmental values | Display flood risk |
| Invalid Input | Incorrect values | Show error message |
| Application Loading | Open website | Display homepage |

## Testing Result
The system successfully performs flood prediction and provides expected outputs.
