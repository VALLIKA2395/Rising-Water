# Project Development Phase

## Introduction
The development phase focuses on implementing the designed system using programming languages and machine learning techniques.

## Development Process

### Dataset Preparation
The flood dataset is collected and processed by removing missing values and preparing suitable input features.

### Model Training
Machine learning algorithms are trained using historical flood data to identify patterns.

### Model Integration
The trained model file is integrated with the Flask application for real-time prediction.

### Application Development
A web application is developed to allow users to enter environmental parameters and receive prediction results.

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML
- CSS

## Output
The system predicts flood risk based on user-provided input values.
